Common HTML Entities

The less-than symbol: < → <
The greater-than symbol: > → >
The ampersand symbol: & → &
Double quote: " → "
Single quote: ' → '

Special Characters

Copyright: © → ©
Trademark: ™ → ™
Registered: ® → ®
Non-breaking space:   → A B C

Why Use Entities?

  • To display characters that have special meaning in HTML
  • To avoid errors or confusion in code
  • To insert symbols not available on the keyboard
