This is an H1 Heading

H1 is the most important heading and is usually used for the page title.

This is an H2 Heading

H2 is used for major sections within the page.

This is an H3 Heading

H3 can be used for sub-sections under H2.

This is an H4 Heading

H4 is a deeper subheading, rarely used unless needed.

This is an H5 Heading

H5 is rarely used, mostly in structured documents.

This is an H6 Heading

H6 is the smallest and least important heading.
